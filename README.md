# 🌿 SmartLeaf: Plant Nutritional Deficiency Detection

## 📌 Overview

SmartLeaf is a Machine Learning-based web application that detects plant nutritional deficiencies using leaf images. The system analyzes leaf patterns and predicts possible nutrient deficiencies to help farmers and agricultural professionals take timely action.

---

## 🚀 Features

* 🌱 Detects plant nutrient deficiencies from leaf images
* 🤖 Uses Deep Learning models for accurate prediction
* 📊 Provides detailed disease information and recommendations
* 🌐 User-friendly web interface for easy interaction
* 📈 Achieves high accuracy (~96%) through optimized models

---

## 🛠️ Tech Stack

* **Programming Language:** Python
* **Machine Learning:** TensorFlow, Keras
* **Libraries:** NumPy, Pandas, OpenCV
* **Frontend:** HTML, CSS
* **Backend:** Flask
* **Tools:** Jupyter Notebook

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/pollakrishnavamsi-lab/SmartLeaf.git
cd SmartLeaf
```

### 2️⃣ Create virtual environment

```
python -m venv venv
venv\Scripts\activate   (Windows)
```

### 3️⃣ Install dependencies

```
pip install -r requirements.txt
```

### 4️⃣ Run the application

```
python app.py
```
